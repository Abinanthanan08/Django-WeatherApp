# ⛅ Django Weather App

A simple Django application that fetches real-time weather data from the OpenWeatherMap API based on the user's city input.

## 🌦 Features

- City-wise weather forecast
- Displays:
  - Temperature 🌡️
  - Pressure ⬇️
  - Humidity 💧
  - Wind Speed 🌬️
- Simple and clean UI

## 🛠 Tech Stack

- Django (Python Web Framework)
- OpenWeatherMap API
- HTML/CSS (for templates)
